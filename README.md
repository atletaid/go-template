# go-template sportiva.id

### How To Run This Project

```
#move to directory
cd $GOPATH/src/github.com/sportivaid/

# Clone into YOUR $GOPATH/src
git clone https://github.com/sportivaid/go-template.git

#move to project
cd go-template

# Install Dependencies
dep ensure

# Make File
make

# Run Project
go run main.go

```
